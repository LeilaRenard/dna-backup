# DNA Backup

[![build][build-img]][build-url]

_Deduplicated versioned backups for DNA._

## Requirements

- Go >= 1.16

## Instructions

_Classical go_

```bash
# Build
go build

# Test
go test ./...

# Run
go run . <source-dir> <repository>
```
